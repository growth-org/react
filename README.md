# growth
## 项目说明：
  ### 该项目是一个前端脚手架，采用es6/es7 + react + mobx + webpack架构。
## 技术说明：
  ### es6/es7：最新版的js语法
  ### react：组件化开发解决方案
  ### mobx：数据状态管理解决方案
  ### webpack：静态资源打包解决方案
  ### json-server：模拟后台接口数据
## 开发说明：
  ### 项目下载后，在growth目录中，执行 npm install，开发时执行命令 npm start 如需模拟后台接口，执行命令 npm mock 打上线包执行 npm run build
